# routing/

## 책임

파이프라인 단계별로 어떤 처리 모델을 쓸지, 실패·저신뢰 시 어디로 넘길지 선택한다. 로컬 7B → 상용 LLM 재검토, VLM 보조 여부, 제공자 장애 시 대체 경로를 결정한다. **판정·생성 자체는 하지 않고, 경로만 정한다.**

## 하위 구조

- `model_router` — 단계·입력 특성에 따라 처리 모델 선택 (추출·분류·재검토·생성 단계별)
- `fallback_policy` — 저신뢰·오류·근거 부족·제공자 장애 시 대체 경로·재시도·중단 정책

## 입력

- 단계 식별자 + 입력 특성 + `confidence` 신뢰도 + 제공자 상태

## 출력

- 선택된 모델·제공자 + fallback 결정 (재검토 대상, 대체 경로, 중단 여부)
