# guardrails/

## 책임

AI 출력이 서비스 원칙을 벗어나지 않게 차단한다. 단정 표현(계약 가능·안전·전세사기·적법/위법 확정)과 근거 없는 출력을 막는다. 생성 결과가 규칙 판정을 바꾸거나 공식 근거 없이 행동을 확정하면 차단·수정한다.

## 하위 구조

- 단정 표현 탐지 (안전/위험/사기 점수·법률 결론 등 금지 표현)
- 근거 유무 검증 (RAG 근거 없는 확정 진술 차단)
- 판정 불변 검증 (생성값이 규칙 판정을 변경하지 않는지 확인)

## 입력

- `generation/`·상용 LLM 출력, 연결된 판정·근거

## 출력

- 통과 / 차단 / 수정 표시된 출력 (통과분만 저장)
- 차단 사유 로그
