# app/core/

## 책임

애플리케이션 공통 기반 계층. 설정 로드(`.env`), 공통 오류 정의·핸들러, 보안 유틸
(비밀번호 해싱·토큰 등, 인증 확정 후), 공통 상수. 도메인 로직은 두지 않는다.

## 입출력

- 설정: 환경변수에서 읽는다. 비밀정보를 코드에 작성하지 않는다.
- 오류: 공통 오류 형식 정의 ([`docs/api/error-format.md`](../../../docs/api/error-format.md))
- 보안: 인증·비밀 처리 유틸(라이브러리 확정 후)
