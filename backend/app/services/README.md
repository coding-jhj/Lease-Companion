# app/services/

## 책임

계약 건·분석 실행 흐름 오케스트레이션 계층. 라우트와 저장소·`ai/` 파이프라인 사이에서
비즈니스 로직을 담당한다. 계약 건 생성, 문서 업로드 처리, 추출값 확인·수정 반영,
분석 실행(로컬 7B → 규칙 엔진 → RAG → 상용 LLM) 조율, 결과 조립·저장.

## 경계 (준수)

- AI 추출·규칙·RAG 로직을 재구현하지 않는다. `ai/` 패키지를 호출한다.
- **규칙 엔진 판정을 LLM·로컬 모델이 임의로 변경하지 못하게 한다.**
- RAG 근거가 없으면 `확인 불가`·`확인 필요`로 처리한다.
- 추출값과 생성값을 구분해 저장소에 전달한다.

## 입출력

- 입력: 라우트가 전달한 검증된 요청, 인증된 사용자
- 처리: `ai/` 호출 오케스트레이션, `repositories/`로 영속화, 장시간 작업은 `workers/`에 위임
- 출력: 도메인 결과(판정·근거·질문·행동·체크리스트)를 라우트/스키마로 반환
