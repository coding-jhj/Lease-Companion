# data/model-metadata/

모델 관련 **메타데이터만** 보관하는 위치. **가중치·체크포인트·어댑터 파일은 두지 않는다.**

> `*.safetensors`·`*.gguf`·`*.bin`·`*.pt`·`*.ckpt`·`adapter_model.*`·`**/checkpoints/`는 `.gitignore`로 제외된다. 여기에는 그 파일들을 **설명하는 메타데이터**만 커밋한다.

## 하위 구조

| 폴더 | 내용 |
|------|------|
| `base-models/` | 후보·선정 베이스 모델 메타데이터 (이름·출처·라이선스·크기 등) |
| `adapters/` | 파인튜닝 어댑터 메타데이터 (학습 데이터셋 버전·하이퍼파라미터·산출 위치 참조) |
| `experiment-results/` | 실험·비교평가 결과 기록 (지표·설정·측정일) |

## 형식

- 가중치 파일 자체가 아니라 **참조·설정·결과**만 기록한다(위치는 외부 스토리지 경로/URI로 참조).
