# 공통 오류 응답 형식

> 설계 방향만 기록한다. 확정 필드·코드 체계는 구현 시 확정한다.

## 방향

- 모든 오류는 일관된 구조로 반환한다.
- 개인정보·문서 내용을 오류 메시지·로그에 포함하지 않는다.
- 사용자에게 노출하는 메시지와 내부 디버그 정보를 분리한다.

## 예상 필드 (초안)

| 필드 | 의미 |
|------|------|
| `error.code` | 기계 판별용 오류 코드 |
| `error.message` | 사용자 표시용 메시지 (개인정보 미포함) |
| `error.details` | (선택) 검증 오류 등 필드 단위 정보 |

## 다뤄야 할 오류 범주

- 입력 검증 실패 (형식·크기·필수값)
- 업로드 파일 형식·크기 초과
- 외부 AI 호출 실패·시간 초과·잘못된 출력
- 세션·리소스 없음
- 서버 내부 오류
