## 원칙

- 계층 분리: API → 서비스 → 저장소.
- 모든 입력·업로드 파일 검증.
- 공통 오류 형식 사용 (`error-format.md`).
- 스키마 변경 시 본 문서와 프론트엔드 타입 동기화.
