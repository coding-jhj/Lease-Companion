# 추출 설계

## 역할

멀티모달 LLM으로 계약서·특약(및 PoC에서 등기사항증명서·건축물대장) 이미지에서 핵심 정보를 추출한다.

## 추출 대상

- 임대인, 주소, 보증금, 계약 기간, 주요 특약 (문서 필드 상세: `../data/document-fields.md`)

## 구조화 출력 원칙

- 모든 추출 결과는 구조화 스키마(`ai/src/lease_companion_ai/schemas/`)로 반환한다.
- 추출값(문서에서 읽은 값)만 담는다. 질문·요약 등 생성값과 섞지 않는다.
- 읽을 수 없거나 근거가 부족한 필드는 추측하지 않고 `정보 부족` / `확인 필요`로 표기한다.
- 추출은 판정하지 않는다. 값을 읽어 반환할 뿐이다.
