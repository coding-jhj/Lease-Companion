# RAG 설계

## 역할

공식 법령·표준계약서·공공기관 가이드에서 근거를 검색해, 확인 항목·질문·체크리스트에 출처를 연결한다.

## 판정 분리 원칙

- RAG는 **근거 검색·설명**만 담당한다. 계약 가능·안전·사기·합법 여부를 **판정하지 않는다.**
- 검색된 근거는 사용자가 확인할 수 있도록 출처와 함께 제시한다.
- 근거가 없으면 지어내지 않고 `정보 부족`으로 반환한다.

## 자료 기준

- 공식 출처 우선. 블로그·커뮤니티 자료를 핵심 근거로 쓰지 않는다.
- 자료마다 기관명·문서명·URL·발행일/확인일 기록. (`../data/rag-sources.md`)
