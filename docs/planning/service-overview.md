# 서비스 개요

## 목적

첫 전월세 계약을 준비하는 임차인이 회원 기반 모바일 웹앱에서 계약 건과 문서를 관리하며, 계약서·특약·등기를 이해하고, 무엇을 확인·질문하고 계약 직후 어떤 권리를 확보해야 하는지 스스로 판단하도록 돕는다.

## 대상 사용자

- 첫 전월세 계약을 준비하는 2030 청년 임차인

## 대상 계약

- 주거용 전세
- 보증부 월세
- 일반 월세

## 최신 MVP 한 줄

회원 기반 모바일 웹앱에서 사용자가 계약 건과 계약 문서를 관리하고, 상용 LLM(Gemini 3.5 Flash)이 임대차 조항 유형과 불명확성 후보를 구조화한다. Python 규칙 엔진은 계약서와 관련 문서의 값을 교차검증하고, 공식자료 RAG와 상용 LLM(GPT-5.6 Sol)은 판정 근거·쉬운 설명·확인 질문·사용자 행동을 생성한다. 파인튜닝한 로컬 7B 모델은 상용 vs 로컬 성능비교 병렬 실험(선택)으로만 유지하며 MVP 크리티컬 패스에서 제외한다.

## 포함 범위

- 회원·계약 건·문서 관리 (계약 건 단위 저장·재조회)
- 계약서·특약(필수)·등기사항증명서·중개대상물 확인설명서(선택) 업로드
- 디지털 PDF 추출(PyMuPDF·PDF.js)·OCR(상용 LLM Gemini 3.5 Flash VLM 통합)로 핵심 정보 구조화, 사용자 확인·수정
- 상용 LLM(Gemini 3.5 Flash) 조항 유형·명확성 후보 구조화 (선택: 로컬 7B 성능비교 실험 — MVP 크리티컬 패스 제외)
- Python 규칙 엔진 문서 내부 판정·교차검증 (4영역 12항목)
- 공식 법령·공공자료 RAG 근거
- 상용 LLM 쉬운 설명·질문·행동 생성, 저신뢰 결과 재검토
- 판정·원문 증거·공식 근거·질문·행동 리포트, 체크리스트·계약 직후 행동 상태 관리

## 제외 범위

- 부동산 매매 / 상가·사무실 임대차 / 경매·공매 / 소송
- 계약 종료 및 보증금 반환 분쟁 / 거주 중 하자 대응

## 핵심 원칙

AI는 계약 가능·안전·전세사기·법률적 결론을 **단정하지 않는다.** 사용자가 직접 확인할 항목·질문·체크리스트·증빙 행동을 제공한다. 판정은 [공통 결과 상태 9개 + 시급도 5개](../data/judgment-spec.md)로 표현하며 `안전`·`위험`·`사기 가능성 점수` 같은 종합 판정은 사용하지 않는다.
