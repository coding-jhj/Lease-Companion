# MVP 범위

> MVP는 사용자에게 제공하는 최소 기능 흐름이다. PoC 기술 검증 범위(`poc-scope.md`)와 혼용하지 않는다.

## 사용자 기능 흐름

1. 현재 계약 단계 선택
2. 계약서·특약 이미지 업로드
3. AI 핵심 정보와 주요 특약 추출
4. 사용자가 추출 결과 확인 및 수정
5. 규칙 엔진이 누락·모호한 조건·확인 필요 항목 분류
6. 확인 질문과 서명 전 체크리스트 제공
7. 계약 직후 행동과 결과 요약 제공

## 포함 범위

- 계약 단계 입력
- 계약서·특약 이미지 업로드
- 핵심 정보·주요 특약 추출
- 추출 결과 사용자 확인·수정
- 확인 필요 항목 분류
- 질문·서명 전 체크리스트
- 계약 직후 행동·결과 요약

## 제외 범위

- 부동산 매매, 상가 임대차, 경매, 소송
- 계약 종료·보증금 반환 분쟁, 거주 중 하자 대응
- 계약 안전·전세사기 여부 단정
- (미확정) 인증, 결제, 알림 등 부가 기능

## 비고

- MVP 입력은 계약서·특약 이미지 중심. 등기사항증명서·건축물대장 자동 비교는 PoC 범위.
