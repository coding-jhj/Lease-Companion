# 사용자 흐름 (기준 6단계)

프로젝트 전체가 따르는 기준 흐름이다.

## 1. 현재 계약 단계 선택

- 계약서 확인 중
- 서명 직전
- 계약 직후

## 2. 계약서·특약 이미지 업로드

## 3. AI 핵심 정보 추출

- 임대인
- 주소
- 보증금
- 계약 기간
- 주요 특약

## 4. 확인 필요 항목 분석

Python 규칙 엔진이 다음을 분류한다.

- 누락 항목
- 모호한 조건
- 문서 간 불일치
- 추가 확인이 필요한 항목

## 5. 질문과 서명 전 체크리스트 제공

- 임대인에게 물어볼 질문
- 공인중개사에게 물어볼 질문
- 우선 확인 항목
- 서명 전 체크리스트

## 6. 계약 직후 행동과 결과 요약 제공

- 전입신고
- 확정일자
- 계약서 보관
- 이체내역 보관
- 필요한 권리 확보 행동
- 분석 결과 요약

## 설계 원칙

AI가 추출한 정보는 **분석(4단계) 전에 사용자가 확인·수정**할 수 있어야 한다.
