# 문서 필드 정의 (초안)

> 문서별 주요 추출 필드 초안. 확정 스키마는 `data/schemas/`에 별도 정의한다.

## 계약서 / 특약

- 임대인 (성명)
- 임차인 (성명)
- 목적물 주소
- 보증금
- (월세인 경우) 차임 / 관리비
- 계약 기간 (시작일·종료일)
- 주요 특약 항목

## 등기사항증명서 (PoC 범위)

- 소유자
- 소재지 주소
- (근)저당권 등 권리관계 항목

## 건축물대장 (PoC 범위)

- 소재지 주소
- 용도
- 위반건축물 여부

## 정규화 대상

- 주소, 금액, 날짜, 이름 → 문서 간 비교 가능한 형태로 정규화 (`ai/normalization`)

## 원칙

- 추출값과 생성값을 구분한다.
- 근거 부족 필드는 `정보 부족`으로 표기한다.
