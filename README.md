# 슬기로운 계약생활 (lease-companion)

첫 전월세 계약을 준비하는 **2030 청년 임차인**을 위한 계약 확인 도우미.
계약서·특약 이미지를 올리면 AI가 핵심 정보를 추출하고, 규칙 엔진이 확인 필요 항목을 분류하며, 공식 자료 RAG로 물어볼 질문·서명 전 체크리스트·계약 직후 행동을 제공한다.

> AI는 계약 가능 여부, 안전 여부, 전세사기 여부, 법률적 결론을 **단정하지 않는다.** 사용자가 직접 확인할 항목·질문·체크리스트·증빙 행동을 제공한다.

## 대상 사용자

- 첫 전월세 계약을 준비하는 2030 청년 임차인
- 대상 계약: 주거용 전세 / 보증부 월세 / 일반 월세

## 해결하려는 문제

첫 계약자는 계약서·특약의 무엇을 확인해야 하는지, 어떤 질문을 해야 하는지, 계약 직후 무슨 권리를 확보해야 하는지 모른다. 정보는 흩어져 있고 판단은 어렵다.

## 핵심 기능

- 계약서·특약 이미지에서 핵심 정보(임대인·주소·보증금·계약 기간·주요 특약) 추출
- 누락·모호·문서 간 불일치·추가 확인 항목 분류
- 임대인·공인중개사에게 물어볼 질문 생성
- 서명 전 체크리스트 제공
- 계약 직후 권리 확보 행동(전입신고·확정일자·서류 보관 등) 안내
- 분석 결과 요약

## 사용자 흐름 (6단계)

1. 현재 계약 단계 선택 (계약서 확인 중 / 서명 직전 / 계약 직후)
2. 계약서·특약 이미지 업로드
3. AI 핵심 정보 추출
4. 확인 필요 항목 분석
5. 질문과 서명 전 체크리스트 제공
6. 계약 직후 행동과 결과 요약 제공

추출 정보는 분석 전에 사용자가 확인·수정할 수 있다. 상세: [docs/planning/user-flow.md](docs/planning/user-flow.md)

## PoC 범위

계약서·등기사항증명서·건축물대장 입력 → 멀티모달 LLM 추출 → 필드 정규화 → Python 규칙 엔진 문서 간 비교 → 공식 법령·가이드 RAG → 확인 항목·질문·체크리스트 생성.
상세: [docs/planning/poc-scope.md](docs/planning/poc-scope.md)

## MVP 범위

계약 단계 선택 → 이미지 업로드 → AI 추출 → 사용자 확인·수정 → 규칙 분류 → 질문·체크리스트 → 계약 직후 행동·요약.
상세: [docs/planning/mvp-scope.md](docs/planning/mvp-scope.md)

## 전체 시스템 구성

```
frontend  →  backend (FastAPI)  →  ai (추출·정규화·규칙·RAG·생성·guardrail)
                                        ↑
                                     data (샘플·스키마·규칙·RAG 자료·평가)
```

상세: [docs/architecture/system-architecture.md](docs/architecture/system-architecture.md)
